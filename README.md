# Automated Essay Scoring

## Requirements
- Tensorflow 1.9
- Keras 2.2.2
- Django 2.1
- Gensim 3.5

```
pip install tensorflow
pip install keras
pip install django
pip install gensim

```

## Installation 
- Clone the repo
- Just run the **Jupyter Notebook** to train the model.
- To run the Django App cd ./ into the **mysite** folder and run ```python manage.py runserver```
